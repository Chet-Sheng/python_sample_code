#### Build up Environment
```bash
pipenv install
# Under Python_Code directory
```

#### Manage Kernels
```bash
bash pipenv_jupyter.sh
```

#### Install table of content in jupyter lab
```bash
jupyter labextension install @jupyterlab/toc
```
